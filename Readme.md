### Pre-requis
```
sudo apt-get install  libreadline-dev
```
### Compile 
```
$ make
```
### Run server 
```
$ ./server -p <port1> -P <port2> -h <server_name>
```
### Run client 
```
$ ./client -p <port1> -P <port2> -h <server_name>
$ > cmd params
```

### Examples

```
$ ./server -p 1234 -P 5678 -h localhost
```

```
$ ./client -p 1234 -P 5678 -h localhost
$ > ls
$ > lls
$ > get ~/emery.pdf
```
